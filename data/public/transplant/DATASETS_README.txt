Transplant Process Intelligence — Data Sources

NMP Trial Summaries (included):
- nmp_vittal_summary.json: VITTAL trial summary (Nasralla et al. Nature 2018)
- nmp_pilot_criteria.json: PILOT viability criteria (Mergental et al. Nat Med 2020)

For full patient-level data, a Data Use Agreement (DUA) is required:

1. UNOS/SRTR (US transplant outcomes):
   - Go to: https://www.srtr.org/requesting-srtr-data/
   - Request SRTR Standard Analysis File (SAF)
   - Required for: donor/recipient demographics, graft outcomes
   - DUA processing time: 4-8 weeks

2. UK Transplant Registry (NHSBT):
   - Go to: https://www.odt.nhs.uk/statistics-and-reports/
   - Request through NHSBT Data Request process
   - Required for: NMP trial patient-level data

3. Eurotransplant:
   - Go to: https://www.eurotransplant.org/professionals/statistics/
   - Contact: research@eurotransplant.org

EAD Definition (Olthoff et al. Liver Transplantation 2010):
- Bilirubin >= 10 mg/dL on day 7
- INR >= 1.6 on day 7
- AST or ALT > 2000 IU/L within first 7 days
